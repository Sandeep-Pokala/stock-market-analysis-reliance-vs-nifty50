# Stock Market Analysis: Reliance vs NIFTY 50

## Overview
This project compares the performance and risk characteristics of Reliance Industries against the NIFTY 50 benchmark using historical market data. The analysis focuses on return behavior, volatility patterns, and long-term performance using a structured analytics workflow.

## Objectives
Compare stock-level performance against a market benchmark
Analyze return and volatility behavior over time
Validate analytical results across Python and SQL
Present insights through an interactive Power BI dashboard

## Data
Historical daily price data for Reliance Industries
Historical daily index data for NIFTY 50
Fields include open, high, low, close prices and trading volume
Raw and processed datasets are organized separately to maintain a clean and reproducible workflow.

## Project Workflow
Data Cleaning & Preparation
Date alignment and data type standardization
Validation of missing values and consistency
Feature Engineering
Daily returns
30-day rolling volatility
Cumulative returns

## Exploratory Data Analysis (EDA)

Return distribution comparison
Volatility trend analysis
Volume and price movement exploration

## SQL Validation

Cross-validation of key metrics using analytical SQL queries
Monthly aggregations and volatility comparison

## Power BI Dashboard

KPI cards for returns and volatility
Cumulative returns comparison
Rolling volatility trends with date-based filtering

## Tools & Technologies

Python: Data cleaning, feature engineering, EDA
SQL: Analytical validation and aggregation
Power BI & DAX: Interactive visualization and reporting

## Key Insights

Reliance exhibits higher return potential compared to the NIFTY 50 but with increased volatility
The NIFTY 50 shows relatively stable performance with lower risk
Rolling volatility highlights periods of heightened market stress and stock-level risk
